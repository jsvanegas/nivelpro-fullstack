Comandos de Git

git init: Inicia un nuevo repo
git add: Agrega los archivos sin trackear al Stage
git commit -m 'msg' : Confirmar cambios
git log: Muestra un log de los commits de la rama
git remote -v : Consultar la lista de remotos
git push: Sube cambios a la rama remota
git pull: Descarga cambios de la rama remota
git log --oneline: Muestra los logs en linea sin mucho texto
git branch -al: Muestra todas las ramas del repo
